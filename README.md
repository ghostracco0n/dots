# My dotfiles

This contains my dotfiles.

## Requirements

Make sure you installed these:

### Git

`pacman -S git`

### Stow

`pacman -S stow`

## Instalation

First, check out the repo in your $HOME directory using git:

`$ git clone https://github.com/ghostracco0n/dots.git`

`$ cd dots`

Then use GNU Stow to create the symlinks

`stow .`
